┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 1. ALTURA DA VOZ (PITCH/TOM) - Qual nota cantar                                                     │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • F0 Médio: 391.0 Hz
   • Range: 133.1 Hz a 1064.8 Hz

🎯 O QUE FAZER:
   ✓ Esta é uma VOZ AGUDA (Tenor/Soprano)
   ✓ Cante na região de D4-F4 (Dó4 a Fá4)
   ✓ Use MAIS voz de CABEÇA/MISTA
   ✓ Leve o som 'para cima' e 'para frente'

💡 COMO PRATICAR:
   1. Use um afinador de celular (apps: Tuner, Pano Tuner)
   2. Cante uma nota sustentada tentando manter 391 Hz
   3. Ajuste até o afinador mostrar a frequência correta


┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 2. VIBRATO - A 'tremidinha' na voz                                                               │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • Presente: SIM
   • Taxa: 3.0 Hz (oscilações por segundo)
   • Extensão: 251.0 cents

🎯 O QUE FAZER:
   ✓ Vibrato LENTO: ondulações lentas e amplas
   ✓ Faça cerca de 3 oscilações por segundo
   ✓ Vibrato INTENSO: variação grande e expressiva

💡 COMO PRATICAR:
   1. Cante uma nota longa e sustentada
   2. Faça pequenas oscilações no tom, 3 vezes por segundo
   3. Use o diafragma: empurre levemente a barriga para dentro/fora no ritmo
   4. OU relaxe a garganta e deixe oscilar naturalmente


┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 3. FORMANTES - Como moldar a 'COR' da voz (timbre único)                                          │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • F1: 690 Hz
   • F2: 1974 Hz
   • F3: 2939 Hz
   • F4: 4042 Hz
   • F5: 4764 Hz

🎯 O QUE FAZER:
   F1 (690 Hz) - ABERTURA DA BOCA:
      ✓ Boca MODERADAMENTE aberta
      ✓ Abertura natural, confortável

   F2 (1974 Hz) - POSIÇÃO DA LÍNGUA:
      ✓ Língua em posição NEUTRA/MÉDIA
      ✓ Som equilibrado

   F3-F5 - RESSONÂNCIA GERAL:
      ✓ Som BRILHANTE/CLARO
      ✓ Mais ressonância na MÁSCARA FACIAL
      ✓ Direcione o som 'para frente' e 'para cima'

💡 COMO PRATICAR:
   1. Grave sua voz e compare com a original
   2. Experimente diferentes posições de língua
   3. Mude a abertura da boca
   4. Use um espelho para observar


┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 4. BREATHINESS - Quanto AR passa pela voz                                                        │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • Nível: ALTO
   • Ratio: 1.180

🎯 O QUE FAZER:
   ✓ VOZ COM MUITO AR (tipo Billie Eilish, Norah Jones)
   ✓ Deixe as cordas vocais MENOS fechadas
   ✓ Permita que MAIS ar escape enquanto canta
   ✓ Som 'sussurrado', íntimo

💡 COMO FAZER:
   1. Não force o fechamento da garganta
   2. Cante como se estivesse sussurrando
   3. Ouça o 'shhh' do ar junto com a nota


┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 5. BRILHO - Som claro/escuro                                                                     │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • Nível: BRILHANTE
   • Centróide: 3911 Hz

🎯 O QUE FAZER:
   ✓ VOZ CLARA/BRILHANTE (muita energia nos agudos)
   ✓ Direcione o som para a MÁSCARA FACIAL

💡 COMO FAZER:
   1. Imagine o som saindo pelo nariz/olhos
   2. Sinta vibração na face (maçãs do rosto)
   3. Som 'apontado', projetado


┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 6. TENSÃO VOCAL - Voz apertada ou relaxada                                                       │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • Nível: BAIXA
   • Ratio: 0.132

🎯 O QUE FAZER:
   ✓ VOZ RELAXADA/ABERTA
   ✓ Garganta BEM aberta
   ✓ Som 'solto', natural

💡 COMO FAZER:
   1. Relaxe TODA a garganta
   2. Como um bocejo: garganta aberta
   3. Som 'fácil', fluido


┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 7. ATAQUE - Como começar cada nota                                                              │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • Attack Time: 13.2005 segundos

🎯 O QUE FAZER:
   ✓ ATAQUE SUAVE/LENTO
   ✓ Notas começam GRADUALMENTE

💡 COMO FAZER:
   1. Comece a nota SUAVEMENTE
   2. Vá CRESCENDO aos poucos
   3. Como uma 'fade in' na voz


┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 8. DINÂMICA - Variações de volume (forte/fraco)                                                  │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • Range Dinâmico: 80.0 dB

🎯 O QUE FAZER:
   ✓ MUITA VARIAÇÃO de volume
   ✓ Cante partes MUITO FORTES e partes MUITO FRACAS

💡 COMO FAZER:
   1. Identifique onde a pessoa canta FORTE
   2. Identifique onde canta FRACO
   3. Exagere o contraste


┌──────────────────────────────────────────────────────────────────────────────────────────────────┐
│ 9. PORTAMENTOS - Deslizes entre notas                                                            │
└──────────────────────────────────────────────────────────────────────────────────────────────────┘

📊 DADOS TÉCNICOS:
   • Portamentos: 312

🎯 O QUE FAZER:
   ✓ MUITOS DESLIZES (estilo melismático)
   ✓ Escorregue/deslize entre QUASE TODAS as notas

💡 COMO FAZER:
   1. NÃO pule direto de uma nota para outra
   2. DESLIZE suavemente
   3. Como uma sirene: uuuuuuuuuu
